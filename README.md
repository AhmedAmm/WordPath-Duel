# WordPath-Duel

A python-based ai search game built with Pygame.

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/AhmedAmm/WordPath-Duel
   cd WordPath-Duel
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python main.py
   ```

## Requirements
- Python 3.7+
- Pygame 2.6.1
